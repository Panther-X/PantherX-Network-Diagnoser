# Server Check (Run on Windows only)
1. Make sure your PantherX and your PC (OS: Windows) in the same network
2. click the 'server-check.bat', it will check the status of PantherX server

About the check result:
If it shows "Unable to open connection:", please check the firewall settings of your network
If it shows "Connect to xxx.xxx.xxx.xxx", it means your connection is good, the PantherX should be also connect to the server well, it not, please check the cable connection in your PantherX
